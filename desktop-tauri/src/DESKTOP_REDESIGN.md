# 桌面端重构设计文档

## 核心问题诊断

**现状**：11 个平级屏幕挤在侧边栏，信息架构扁平，视觉混乱
- planning（策划）
- chat（探讨）  
- studio（创作）
- sessions（会话历史）
- chapters（章节）
- characters/foreshadow/settings（3个记忆库）
- collab（协作）
- checkpoints（快照）
- tools（工具）
- providers（供应商）

用户需要在这 11 个平级入口中跳转，缺乏清晰的工作流引导。

## 新架构：书桌工作流

### 隐喻：书桌上的三叠宣纸

想象一个传统书桌，上面摊开三叠重叠的宣纸，你每次只激活一张来工作，其他两张半透明地收在旁边：

```
┌─────────────────────────────────────────────────┐
│  [策划]    [创作·]    [修订]         [⚙️设置] │  ← 顶部横向标签
├─────────────────────────────────────────────────┤
│                                          [抽屉] │
│  ┌───────────────────────┐              │人物│ │
│  │ 面板：构思            │              │库 │ │
│  │ (输入框+建议)          │              │   │ │
│  ├───────────────────────┤              │会话│ │
│  │ 面板：与 AI 探讨      │              │   │ │
│  │ (对话界面)            │              │协作│ │
│  ├───────────────────────┤              └────┘ │
│  │ 面板：生成设定        │                     │
│  │ (人物/伏笔/设定生成)   │                     │
│  └───────────────────────┘                     │
│          ↕ 纵向滚动                            │
└─────────────────────────────────────────────────┘
```

### 三个主标签（Work Tabs）

#### 1. 策划（Planning）
**用途**：写作前的构思和世界观建设
**纵向面板**（垂直滚动）：
- 📝 构思：textarea 输入想法 + AI 提供起聊建议
- 💬 探讨：与 AI 多轮对话，精炼想法（整合旧 chat screen）
- 🎨 生成设定：一键让 AI 生成人物/伏笔/世界观并入库（整合旧 planning）

#### 2. 创作（Studio）
**用途**：运行 AI agent 生成章节
**纵向面板**：
- 🎯 目标：输入创作任务（"写第三章：主角与反派初次对峙"）
- ⚙️ 控制台：工作状态 + 工作流程 + 实时 agent 馈给（推理/工具调用）
- 📄 成稿预览：生成的内容实时显示

#### 3. 修订（Revision）
**用途**：编辑已生成的章节
**纵向面板**：
- 📚 章节列表：左侧章节卡片，点击加载
- ✏️ 编辑器：右侧大文本区域 + 工具栏（保存/删除/快照）

### 辅助抽屉（Side Drawers，从右侧滑入）

不再是主导航的一部分，而是**按需弹出的参考面板**：

#### 人物库抽屉
- 卡片网格展示所有角色
- 点击卡片展开详情浮层
- 右上角搜索框 + 标签筛选

#### 会话历史抽屉
- 按时间倒序列出所有创作/探讨会话
- 点击「继续」跳转到对应标签并载入会话

#### 协作抽屉
- 快照列表（checkpoints）
- 提交历史（collab commits）
- 分支管理

### 设置区（独立入口）

**触发方式**：右上角齿轮图标 → 全屏遮罩弹出设置面板
**内容**：
- 供应商配置
- 工具目录
- 应用偏好

## 视觉语言：克制的书桌

### 布局特征
- **顶部横向标签**：3个主标签 + 右侧设置按钮，固定 48px 高
- **活跃区域**：单一纵向滚动容器，不套卡片，用朱砂分隔线区分面板
- **右侧抽屉**：420px 宽，从右滑入，半透明遮罩 + ESC 关闭
- **不再有侧边栏**：删除整个 Sidebar 组件

### 配色与排版
- **保持水墨色系**：宣纸 #efe8da、墨 #25201c、朱砂 #b73225
- **标签激活态**：底部 3px 朱砂下划线 + 字重加粗
- **面板分隔**：1px 朱砂虚线 `border-top: 1px dashed var(--cinnabar)`
- **抽屉背景**：`rgba(239, 232, 218, 0.98)` 带轻微模糊
- **字体**：
  - 标题/标签：Songti SC（宋体）
  - 正文：PingFang SC
  - 代码/工具输出：Cascadia Code

### 动效
- 标签切换：内容区淡出淡入 0.28s
- 抽屉：从右滑入 0.32s cubic-bezier(0.22, 1, 0.36, 1)
- 面板展开/收起：grid-template-rows 过渡
- 全程尊重 prefers-reduced-motion

## 组件清单

### 新增组件
- `WorkTabs.tsx`：顶部横向标签组
- `Panel.tsx`：纵向面板容器（带标题/可折叠）
- `Drawer.tsx`：右侧抽屉容器（带遮罩/滑入动画）
- `SettingsModal.tsx`：全屏设置面板

### 重写屏幕
- `PlanningWork.tsx`：策划标签内容（构思+探讨+生成）
- `StudioWork.tsx`：创作标签内容（agent 运行）
- `RevisionWork.tsx`：修订标签内容（章节列表+编辑器）

### 保留并重构
- `TitleBar.tsx`：保持无边框标题栏
- `Toast.tsx`：保持通知系统
- agent 相关组件（WorkStatus/ReasoningBlock/ToolCallCard/Conversation）：保留并复用

### 删除组件
- `Sidebar.tsx`：整个删除
- `EmptyState.tsx`：简化为内联占位符

## 迁移路径

### 旧屏幕映射到新架构

| 旧屏幕 | 新位置 |
|--------|--------|
| planning | 策划标签 → 生成设定面板 |
| chat | 策划标签 → 探讨面板 |
| studio | 创作标签 → 整个内容 |
| chapters | 修订标签 → 整个内容 |
| characters/foreshadow/settings | 人物库抽屉（合并为统一的记忆库） |
| sessions | 会话历史抽屉 |
| collab + checkpoints | 协作抽屉（合并） |
| tools + providers | 设置区（合并） |
